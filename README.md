# Student Performance Prediction – Jnanasetu

## Overview
Jnanasetu is a Machine Learning based Student Performance Prediction System designed to analyze academic factors such as study hours, attendance, and previous scores to predict student exam performance. The project helps teachers identify students who may require additional academic support and enables students to better understand and improve their learning habits.


## Features
- Student performance prediction using Machine Learning
- Data preprocessing and analysis
- Correlation heatmap visualization
- Model training and evaluation
- Interactive and user-friendly web interface
- Instant prediction results with visual feedback



## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- Streamlit



## Dataset
- Dataset sourced from Kaggle
- Includes factors such as:
  - Hours Studied
  - Attendance
  - Previous Scores
  - Motivation Level
  - Parental Involvement



## Project Workflow
1. Data Collection  
2. Data Cleaning & Preprocessing  
3. Exploratory Data Analysis  
4. Correlation Heatmap Visualization  
5. Model Training & Testing  
6. Frontend Development using Streamlit  
7. Prediction & Result Display
